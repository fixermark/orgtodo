# OrgTodo: Track your TODO list, org-style

OrgTodo is a simple TODO tracker inspired by (and mostly compatible with) emacs org-mode.

## Features

* Synchronize your TODOs back and forth with emacs org-mode
* Sort by priority or by deadline
* Checkbox lists and clicking off individual checkboxes

OrgTodo is particularly designed for

* Shopping lists
* Quick-glance task tracking
